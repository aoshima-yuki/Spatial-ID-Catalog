空間ID関連OSSリスト

GitHub上で公開されている空間ID関連OSSの候補を自動収集し、リストとして表示しています。

概要

本ページは、GitHub上で公開されている空間ID関連OSSの候補を自動収集し、リストとして表示するものです。掲載内容は、空間IDに関するOSSの発見性を高めるための参考情報です。

リスト生成日時：読み込み中...

作成方法

  1. GitHub API を用いて、公開リポジトリからキーワードに一致する候補を収集します。
  2. リポジトリ検索に加え、README等のMarkdownファイル検索からも候補を収集します。
  3. 候補リポジトリごとにメタデータとREADMEを取得し、検索・判定対象項目にキーワードが含まれるかを確認します。
  4. 除外リストに登録されたリポジトリを除外し、Stars数及び更新日時を基準に並べ替えてリストデータを生成します。

検索・判定対象項目

  • リポジトリ名（name）
  • リポジトリ概要（description）
  • Topics
  • README
使用キーワード： 空間ID, 空間ＩＤ, spatial-id, spatialid

英字の大文字・小文字は区別せずに判定します。

注意事項

  • 本リストはキーワードに基づく自動収集により生成しています。
  • 掲載されているリポジトリは、評価・推奨・認定を意味するものではありません。
  • キーワード検索の特性上、空間IDと直接関係しないリポジトリが含まれる場合があります。
  • GitHub API の仕様や検索条件により、すべての関連OSSを網羅できるとは限りません。
